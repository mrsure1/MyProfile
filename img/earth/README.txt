로컬 Spline 3D 배경 테스트용 폴더입니다.

현재 img/earth.spline 파일은 Spline 편집용 프로젝트 파일이라 브라우저에서 직접 재생되지 않습니다.

사용 방법:
1. Spline 앱에서 earth.spline 열기
2. 상단 Export → "Web" 또는 "Export as Code" 선택 후 다운로드(zip)
3. zip 압축 해제한 내용을 이 폴더(img/earth/)에 넣기
4. 반드시 index.html 이 이 폴더에 있어야 합니다 (예: img/earth/index.html)

그 후 메인/포트폴리오 페이지를 새로고침하면 로컬 지구본이 로드됩니다.
내보내기 파일이 없으면 자동으로 원격 Spline URL로 전환됩니다.
